# GPUプログラミング入門

東京大学情報基盤センター お試しアカウント付き講習会「GPUプログラミング入門」で使用しているOpenACC(C, Fortran)のサンプルコードです。  
OpenACCの基本である、kernels, loop, data指示文の適用方法が学べます。  
スーパーコンピュータMiyabi-G向けのジョブスクリプトが含まれます。
